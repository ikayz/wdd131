How It Works

Getting started is simple — BudgetNest walks you through each step with helpful prompts and clear visual feedback. Below is a recommended 10–15 minute setup to get value quickly.

  1. Create an account — choose an email and password. Optionally connect your bank using a read-only connector (secure and encrypted). If you prefer, skip bank linking and enter transactions manually.
  2. Import or add transactions — connect to your account or import a CSV. BudgetNest will suggest categories for each transaction. Review a few and correct as needed so the system learns your preferences.
  3. Set budgets and goals — start with high-level categories (Housing, Food, Transport). Try a small savings goal first (e.g., 10% of income) to see quick progress.
  4. Use insights — Weekly summaries highlight where you spent most, recurring charges, and suggestions like “Cut dine-out by 15% to meet your goal sooner.” Apply suggestions or adjust budgets directly.
  5. Iterate — budgets are not set-and-forget. Review monthly, adjust categories, and let the app reallocate leftover amounts to savings automatically.

Tips for success

  • Start small: pick one category to actively manage this month.
  • Use rules to auto-categorize recurring transactions (subscriptions, rent).
  • Schedule a 10-minute weekly review to keep budgets realistic.
