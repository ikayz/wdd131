BudgetNest Site Plan

A personal finance management tool website

Site Name

BudgetNest – The name reflects a safe, organized space (“nest”) for managing and growing personal finances.

Optional domain: budgetnest.com

Site Purpose

The purpose of the site is to introduce BudgetNest and help users understand how it can support their financial goals. The website will provide feature highlights, explain how the app works, and allow users to contact the developer for more information.

Scenarios

  • How can I track my monthly expenses and savings goals using BudgetNest?
  • Where can I learn more about the features before downloading the app?

Color Schema

The site will use a clean, professional palette:

  • Primary color: #2b6cb0 (used for headings and accents)
  • Secondary color: #f6ad55 (used for buttons and highlights)
  • Background color: #f9fafb (for a light, clean background)

Typography

  • Poppins – for headings, giving a modern, professional look.
  • Arial – for body text, ensuring readability and simplicity.
